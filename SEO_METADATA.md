## Cấu Trúc Dữ Liệu Schema.org (Structured Data)

### Organization Schema
Được sử dụng trong tất cả các trang để định nghĩa thông tin công ty:

```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Công ty Cổ phần Công nghiệp Đại Phong",
  "alternateName": "Đại Phong",
  "legalName": "Công ty Cổ phần Công nghiệp Đại Phong",
  "url": "https://www.onggiodaiphong.com",
  "logo": "https://www.onggiodaiphong.com/images/logo/logo.png",
  "telephone": "+84888885964",
  "email": "ongiodaiphong@gmail.com",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "Lô 3.2, Đường số 2, KCN Tân Đông Hiệp A, Phường Dĩ An",
    "addressLocality": "Thành phố Hồ Chí Minh",
    "addressCountry": "VN"
  }
}
```

### LocalBusiness Schema (Đề xuất thêm)
Nên thêm vào trang chủ và trang liên hệ để tối ưu Local SEO.

### Product Schema (Đề xuất thêm)
Nên thêm vào từng trang sản phẩm chi tiết.

Đã được thêm vào các trang tin tức và hoạt động công ty.

## Hướng Dẫn Cập Nhật Metadata

### Khi Thêm Trang Mới

1. **Tạo metadata trong page.tsx hoặc layout.tsx**
```typescript
export const metadata: Metadata = {
  title: 'Tiêu đề trang - Đại Phong',
  description: 'Mô tả ngắn gọn về trang...',
  keywords: 'từ khóa 1, từ khóa 2, từ khóa 3',
  // ... các trường khác
};
```

2. **Thêm structured data nếu cần**
```typescript
const structuredData = {
  '@context': 'https://schema.org',
  '@type': 'WebPage',
  // ... thông tin structured data
};
```

3. **Cập nhật sitemap.ts**
Thêm URL mới vào danh sách sitemap.

### Best Practices

1. **Title Tags**
   - Bao gồm từ khóa chính
   - Độ dài: 50-60 ký tự
   - Format: [Từ khóa chính] | [Tên công ty]

2. **Meta Descriptions**
   - Mô tả hấp dẫn, có CTA
   - Độ dài: 150-160 ký tự
   - Bao gồm từ khóa chính và phụ

3. **Keywords**
   - 10-20 từ khóa liên quan
   - Ưu tiên từ khóa có search volume cao
   - Bao gồm cả long-tail keywords

4. **URLs**
   - Ngắn gọn, có ý nghĩa
   - Sử dụng dấu gạch ngang (-)
   - Bao gồm từ khóa chính

## Lưu Ý Quan Trọng

1. **Không Thay Đổi Nội Dung**: Metadata chỉ ảnh hưởng đến SEO, không thay đổi nội dung hiển thị trên trang.

2. **Cập Nhật Định Kỳ**: Nên review và cập nhật metadata mỗi 3-6 tháng dựa trên:
   - Thay đổi của thuật toán Google
   - Hiệu suất từ khóa
   - Feedback từ Google Search Console

3. **Tính Nhất Quán**: Giữ tone & voice nhất quán trên tất cả các trang.

4. **Mobile-First**: Google ưu tiên mobile indexing, đảm bảo metadata hiển thị tốt trên mobile.
